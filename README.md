# Hybrid Cloud Load Balancing Project (Azure)

## Overview
This project demonstrates a hybrid cloud architecture using Microsoft Azure with load balancing, monitoring, and automation.

## Technologies
- Microsoft Azure
- Linux (Ubuntu)
- Azure Load Balancer
- Azure Monitor
- Bash scripting

## Features
- Two web servers running in Azure
- Load balancing between VMs
- Network security rules (NSG)
- Monitoring dashboard and alerts
- Automation scripts for system checks
